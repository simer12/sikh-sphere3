## 🌐 APIs Used:

### Current:
1. **SikhNet API** - For daily Hukamnama
   - Endpoint: `https://api.sikhnet.com/hukamnama`
   - Returns: Gurmukhi, translations, Teeka

### Recommended for Future:
2. **GurbaniNow API** - For complete Gurbani database
   - `https://api.gurbaninow.com/v2/`
   - Search Shabads, Ang-by-Ang Gurbani
   
3. **SikhiToTheMax API** - For advanced Gurbani search
   - `https://api.banidb.com/v2/`
   - Includes full Guru Granth Sahib Ji database

4. **Akhand Paath API** - For continuous Gurbani recitation
   - Real-time audio streams from Golden Temple
   
5. **Google Places API** - For finding nearby Gurdwaras
   - User's actual location-based search
   - Reviews and photos
