## 🚀 How to Run Your App (3 Simple Steps)

### Step 1: Open Terminal Here
In this folder, right-click and select "Open in Terminal" or "Open PowerShell"

### Step 2: Start the App
```powershell
npm start
```

### Step 3: Open on Your Phone
**Option A: Using Expo Go (Easiest)**
1. Install "Expo Go" from Play Store (Android) or App Store (iOS)
2. Scan the QR code shown in your terminal
3. App opens instantly!

**Option B: Using Emulator**
- Press `a` for Android emulator
- Press `i` for iOS simulator (Mac only)

## 🎨 Customization Quick Tips

### Change Colors:
Open `src/theme.ts`:
```typescript
primary: '#FF9933',  // Change to your color
secondary: '#000080', // Change to your color
```

### Add More Banis:
Open `src/data/banis.ts` and add to the `nitnemBanis` array

### Add More Gurdwaras:
Open `src/screens/GurdwaraFinderScreen.tsx` and add locations

### Add More Gurus:
Open `src/data/banis.ts` and expand the `gurus` array

## 🏆 Ready to Deploy?

### Testing First:
```powershell
npm start              # Test locally
```

### Build for Production:
```powershell
npm install -g eas-cli       # Install EAS CLI
eas build --platform android # Build APK
eas build --platform ios     # Build IPA
```

### Publish to Stores:
```powershell
eas submit --platform android # Play Store
eas submit --platform ios     # App Store
```
